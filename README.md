# Hospital Readmission Prediction using Machine Learning

Saima Furqan

## Project Overview
Hospital readmissions are a major challenge in healthcare systems, increasing costs and indicating potential gaps in patient care. This project uses machine learning to predict the risk of hospital readmission based on patient clinical characteristics.

This project demonstrates healthcare data analytics and predictive modeling skills relevant to improving patient outcomes and healthcare efficiency.

---

## Objectives

- Predict hospital readmission risk using patient data
- Identify key clinical factors contributing to readmission
- Demonstrate machine learning application in healthcare analytics
- Support data-driven decision-making in healthcare systems

---

## Dataset Features

The dataset includes patient clinical features such as:

- Age
- Diabetes status
- Hypertension status
- Readmission outcome (target variable)

---

## Tools and Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Matplotlib
- Seaborn

---

## Machine Learning Model

Random Forest Classifier was used to:

- Train predictive model
- Identify patterns in healthcare data
- Predict patient readmission risk

---

## Workflow

1. Data collection and preprocessing  
2. Exploratory data analysis  
3. Feature selection  
4. Model training using Random Forest  
5. Model evaluation  
6. Prediction of readmission risk  

---

## Results

The machine learning model successfully predicts hospital readmission risk based on patient clinical features, demonstrating the potential of predictive analytics in improving healthcare outcomes.
